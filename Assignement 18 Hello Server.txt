Assignment Name: Hello Server

Objective

To create a simple Node.js server that responds with different messages for different routes.
1. Node.js Code (Basic Server)
Create a file: server.js

const http = require("http");

const server = http.createServer((req, res) => {

  if (req.url === "/") {
    res.write("Welcome to Home Page");
  } 
  else if (req.url === "/about") {
    res.write("This is About Page");
  } 
  else if (req.url === "/contact") {
    res.write("Contact us at: example@gmail.com");
  } 
  else {
    res.write("404 Page Not Found");
  }

  res.end();
});

server.listen(3000, () => {
  console.log("Server running on http://localhost:3000");
});
▶️ How to Run
Install Node.js
Open terminal in project folder
Run:
node server.js
Open browser:
http://localhost:3000/
http://localhost:3000/about
http://localhost:3000/contact
💡 Output
/ → Welcome message
/about → About page message
/contact → Contact info
Other → 404 message
