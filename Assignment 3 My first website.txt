Theory: My First Website
🔹 Introduction

A website is a collection of web pages created using HTML. In this assignment, a personal profile webpage is designed to display basic information such as name, image, skills, and interests.

🔹 Objective

To create a simple webpage using HTML elements like headings, images, links, and lists.

🔹 HTML Elements Used
Headings (<h1> to <h3>)
Used to display titles and section headings.
Paragraph (<p>)
Used to write information about the user.
Image (<img>)
Displays profile picture using the src attribute.
Lists (<ul>, <li>)
Used to display skills and hobbies in bullet points.
Links (<a>)
Used to navigate to other websites.
Div (<div>)
Used to structure and group content.

🔹 CSS Styling
CSS is used to enhance the appearance of the webpage:
Background color is added
Text alignment is set
Image is styled as a circle
Box shadow is used for better design

🔹 Features of the Webpage
Displays personal profile
Includes profile image
Shows skills and hobbies
Contains external link
Simple and user-friendly design

🔹 Advantages
Easy to create and understand
Helps in learning basic web development
Can be expanded into a portfolio
🔹 Conclusion

This assignment helps in understanding the basic structure of a webpage and the use of important HTML elements. It is the first step towards building more advanced websites.
