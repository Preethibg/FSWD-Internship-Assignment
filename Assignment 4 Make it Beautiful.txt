Theory: Make It Beautiful (CSS Styling)
🔹 Introduction
CSS (Cascading Style Sheets) is used to enhance the visual appearance of a webpage. It allows developers to add colors, fonts, spacing, and layout design to HTML elements.

🔹 Objective
To improve the previously created webpage by applying CSS styling such as colors, fonts, borders, and spacing.

🔹 Styling Techniques Used
Colors
Background gradient applied to the page
Text and heading colors for better readability
Fonts
Used modern font (Segoe UI) for clean appearance
Borders
Profile image styled with circular border
Sections separated using lines
Spacing
Margin and padding used for proper layout
Line spacing for readability
Box Shadow
Added shadow to container for depth effect
Hover Effects
Button color changes when mouse is hovered

🔹 CSS Properties Used
background
color
font-family
margin, padding
border, border-radius
box-shadow
transition

🔹 Features of Styled Webpage
Attractive and modern design
Better readability
User-friendly interface
Interactive elements (hover effect)

🔹 Advantages
Improves user experience
Makes website visually appealing
Helps in professional web design

🔹 Conclusion
Styling a webpage using CSS enhances its appearance and usability. This assignment helps in understanding how design plays an important role in web development.
